# Database

- Cơ sở dữ liệu
- Nơi lưu trữ dữ liệu của ứng dụng phần mềm
- Dễ dàng tra cứu, cập nhật (Truy vấn)

2 loại:

- SQL: Cơ sở dữ liệu quan hệ --> MySQL (MariaDB)
- NoSQL: Cơ sở dữ liệu phi quan hệ

Học cú pháp ngôn ngữ truy vấn SQL

```
Database Server
    - Database 1
        - Table 1
            - Field 1 (Column)
                - Record 1
                - Record 2
                - Record 3
            - Field 2
                - Record 1
                - Record 2
                - Record 3
            - Field 3
                - Record 1
                - Record 2
                - Record 3
            - Field n
        - Table 2
        - Table 3
        - Table n
    - Database 2
    - Database n
```
